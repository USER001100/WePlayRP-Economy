1. עריכת מידע תעשה חופשי
2. מחיקת שורות תעשה רק דרכי

4. לא לשגות בשום עריכה לעולם
5. עדיפות תמידית ל- Pull Request כי זה עוזר למנוע טעויות
6. בחיים לא לערוך כל דבר שנקרא "SHORTNAME" בלי שזה עובר דרכי.
